# 汉兜Handle求解器

**请不要运行a.py，否则会覆盖answers_noted.json中u->v的修改**

## 使用方法

运行solver.py，默认为标准模式，可在左上角“游戏模式”中进行更改

在最上方分别输入四个字的“汉字/声母/韵母/声调”信息，分别存在“正确/存在但位置错误/不存在”三种情况，完成后点击“确定”

中间一栏显示了当前条件下出现频率最高的语素，可以在右侧调整推荐算法中四种语素的权重，默认为50/20/20/10（频率均会进行归一化，
因此设置权重时不必考虑语素之间的频率差异），完成后点击“确定”

最下方的保守策略展示了当前剩余词语中频率得分最高的6个词语，最大熵策略展示了所有词语中频率得分最高的6个词语，因此最大熵策略的词语可能违反已有规则

**需要注意特殊情况，若猜测中存在重复的clue，需人工甄别，如下图**
